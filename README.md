О проекте

Codebattle - open-source проект Хекслета и комьюнити.  
Это соревновательная игра, где два программиста одновременно решают одну задачу на разных языках программирования.  
Игроки видят редактор соперника в реальном времени, результаты тестов и прогресс решения. Побеждает тот, кто решит задачу быстрее.

Ссылка на проект: https://codebattle.hexlet.io/

Основные возможности Codebattle:
- Игры 1v1 (против игрока, друга по ссылке или бота)
- Выбор сложности и конкретной задачи или random-задача
- Решение задач на разных языках программирования
- Редактор кода с отображением прогресса соперника
- Чат во время игры
- Возможность наблюдать за игрой
- История игр и запись завершенных матчей
- Система рейтинга (неделя / месяц / всё время)
- Поддержка задач на русском и английском языках

Цель репозитория

Репозиторий создан для практики "тестирования в продакшене" на реальном проекте с активной разработкой и вкладом большого количества участников.

Содержимое репозитория:
- README.md — описание проекта и цели тестирования
- test-cases/ — тест-кейсы и чек-листы (если будут добавлены)
- bug-reports/ — баг-репорты с шагами воспроизведения и ожидаемым результатом
- notes/ — заметки по поведению системы и найденным особенностям
